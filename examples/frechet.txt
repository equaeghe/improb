Calculating the Frechet bounds
==============================

>>> from improb.lowprev import LowPrev
>>> lpr = LowPrev(4)
>>> lpr.setprecise([1,1,0,0], 0.6)
>>> lpr.setprecise([0,1,1,0], 0.7)
>>> lpr.isavoidingsureloss()
True
>>> print "%.6f" % lpr.getlower([0,1,0,0]) # should be max(0.6+0.7-1,0)
0.300000
>>> print "%.6f" % lpr.getupper([0,1,0,0]) # should be min(0.6,0.7)
0.600000
